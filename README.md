# 일정 관리 간트 차트 (Next.js)

담당자, 세부 일정, 메모, 도형/색상 변경 기능을 가진 간트 차트입니다.

## 실행 방법

1. 압축을 풀고 폴더로 이동합니다.
2. 패키지를 설치합니다.

   ```bash
   npm install
   ```

3. 개발 서버를 실행합니다.

   ```bash
   npm run dev
   ```

4. 브라우저에서 `http://localhost:3000` 을 엽니다.

## 참고

- 새로고침하면 데이터가 초기화됩니다 (브라우저 메모리에만 저장).
- 실제 서비스로 운영하려면 `components/GanttChart.tsx` 안의 상태 관리를 데이터베이스나 API 호출로 바꾸면 됩니다.
- 막대를 드래그하면 일정이 이동하고, 양 끝을 잡고 당기면 기간이 늘어나거나 줄어듭니다.
- 막대 위 작은 동그라미 버튼 두 개로 도형(사각형/알약형/마름모)과 색상(무지개 7색)을 바꿀 수 있습니다.
- 메모 아이콘을 누르면 그 작업/세부 일정 아래에 메모칸이 펼쳐지고, 다시 누르면 닫힙니다.
